# Pavel Shpagin - Personal Portfolio

A minimalist brutalist portfolio website built with Next.js, TypeScript, and Tailwind CSS.

## Features

- Black/white brutalist design aesthetic
- Fully responsive layout
- Sections: Education, Publications, Experience, Projects, Skills, Awards
- Social links with icons (GitHub, LinkedIn, LeetCode)
- SEO optimized with meta tags
- Print-friendly styles

## Tech Stack

- **Framework**: Next.js 15
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Icons**: React Icons

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the website.

### Build

```bash
npm run build
```

### Production

```bash
npm start
```

## Deployment

This project is configured for deployment on Vercel:

1. Push to GitHub
2. Import project in Vercel
3. Deploy automatically
